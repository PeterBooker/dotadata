API

You can access data directly from the API, with the following endpoints. All endpoints have a single (optional) query parameter: lang, which allows you to define the language.

Abilities

URL: https://dota.peterbooker.com/api/v1/abilities/

Ability

URL: https://dota.peterbooker.com/api/v1/ability/{name}/

Heroes

URL: https://dota.peterbooker.com/api/v1/heroes/

Heroes

URL: https://dota.peterbooker.com/api/v1/hero/{name}/

Items

URL: https://dota.peterbooker.com/api/v1/items/

Item

URL: https://dota.peterbooker.com/api/v1/item/{name}/
